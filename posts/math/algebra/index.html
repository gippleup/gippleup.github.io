Trigonometric function

Radian

  • The angle for a arc, which has length of r, is called radian.
  • It can be derived like this.
360 : 2 * PI * r = radian * r
=> (2 * PI * r) * radian = 360 * r;
=> radian = 360 * r / (2 * PI * r);
=> radian = 180 / PI;
