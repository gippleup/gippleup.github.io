Key props

flex-wrap

  • Flex wrap prop defines whether to wrap content for multiple lines or not.
  • For this to be useful, contents should have their own width or height. If not, content would just size in their own inner content size.
#main {
  display: flex;
  flex-wrap: wrap;
}
