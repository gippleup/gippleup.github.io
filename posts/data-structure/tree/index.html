Data Structure: Tree

Mar 23, 2020

Hierarchy? combinations? That’s all tree!

Concept

(img from programiz)

Tree data structure can boost access time.

Let’s say, we have 10,000 items in an array and we don’t know where the target item is. When we want to access the item, the minimum time-complexity is O(n) because we need to compare every items in the array.

However, if we can arrange the data in a tree structure, it can reduce access time exponentionally. For example, if every nodes can be structured with 10 children for each. The depth of tree will be 4 and each depth would require only 10 calculation, which would result in 40 or lower calculation even in worst case and 4 calculation in best case.

In ideal setting, 10,000 calculation can be replaced with 4~40 calculation, with tree data structure.


Exmaple of tree structure

  • Taxonomy: Taxonomy usually follows hierarchical order, which implies it can be structured in tree form. See this
  • File system: File systems can be illsurated with tree structure. See this

addChild: Add a child tree node to existing tree. contains: Check if an item exist in the tree.


# Graph, written in JS 
const Tree = function (value) {
  const newTree = Object.create(treeMethods);
  newTree.value = value;

  newTree.children = [];
  return newTree;
};

const treeMethods = {};

treeMethods.addChild = function (...args) {
  for (const value of args) {
    let newNode = new Tree(value);
    this.children.push(newNode);
  }
};

treeMethods.contains = function (target) {
  if (this.value === target) {
    return true;
  } else {
    for (const child of this.children) {
      if (child.value === target) {
        return true;
      }
      if (child.children.length) {
        let result = this.contains.call(child, target);
        if (result === true) {
          return true;
        }
      }
    }
  }
  return false;
};

Advantages

  • Trees can reflect structural relationships in the data.
  • Trees provide an efficient insertion and searching because it allows us to access to item efficiently.

Disadvantages

  • It’s not opt for non-structural data.
