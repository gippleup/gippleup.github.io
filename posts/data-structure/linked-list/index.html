Data Structure: Linked List

Mar 22, 2020

For removing and adding an item, linked list is handy.

Concept

(img from programiz)

Linked list nodes have pointer in its own. What it means simply, node.next = nextNode. That is, you can access to next node by using this code node.next. Items are added to tail node.

Belows are common methods included in linked list.

  1. Add To Tail: Adds an item to the list – Time Complexity : O(1)
  2. Remove Head: Removes an item in the head and return the value. If the queue is empty, then it is said to be an Underflow condition – Time Complexity : O(1)
  3. Contains: Check if an item exist in the list – Time Complexity : O(n)

# Linked List, written in JS 
const LinkedList = function () {
    this.head = null;
    this.tail = null;
}

LinkedList.prototype.addToTail = function (value) {
    let newEle = new Node(value);
    if (this.head) {
        this.tail.next = newEle;
        this.tail = newEle;
    } else {
        this.head = newEle;
        this.tail = newEle;
    }
}

LinkedList.prototype.removeHead = function () {
    let result;
    if (this.head) {
        result = this.head.value;
        this.head = this.head.next;
    } else {
        return;
    }
    return result;
}

LinkedList.prototype.contains = function (value) {
    let result = false;
    this.forEach((node) => {
        if(node.value === value) {
            result = true;
            return 'finish'
        }
    })
    return result;
}

LinkedList.prototype.forEach = function (callback) {
    let curNode = this.head;
    let depth = 0;
    let result;
    while (curNode && result !== 'finish') {
        callback(curNode, depth, this);
        // result = callback(curNode, depth);
        curNode = curNode.next;
        depth += 1;
    }
}

LinkedList.prototype.filter = function (callback) {
    let newList = new LinkedList();
    this.forEach((node, depth) => {
        if (callback(node, depth, this)) {
            newList.addToTail(node.value);
        }
    })
}

LinkedList.prototype.map = function (callback) {
    let newList = new LinkedList();
    this.forEach((node, depth) => {
        let result = callback(node, depth, this);
        newList.addToTail(result);
    })
}

const Node = function (value) {
    this.value = value;
    this.next = null;
}

Advantage of Linked List

It’s easy to insert an item, and easy to access to next item. This can be useful when you have a task which requires you to do tasks in order.

Disadvantage of Linked List

It’s hard to modify items. Items are not marked with keys or name, which means you cannot access to specific item in single step. It forces you to search for all nodes if you want to retrieve an item.
