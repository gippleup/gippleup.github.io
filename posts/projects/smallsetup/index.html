Setup for Small Project

May 30, 2020

//project structure
project
    \__build
    \__node_modules
    \__src
    index.html
    package.json
    webpack.config.js

Intro

  • This setup is for using node modules in simple html. You can achieve this by using webpack and babel.

Steps

  1. Install webpack and babel
npm i -D webpack webpack-cli webpack-dev-server babel-loader babel-preset-env @babel/core @babel/preset-env @babel/polyfill
  1. Add webpack.config.js in your root folder
const path = require('path');

module.exports = {
  entry: {
    app: [
      "@babel/polyfill",
      "./src/app.js"
    ]
  },
  output: {
    path: path.resolve(__dirname, "build"),
    filename: "./bundle.js"
  },
  module: {
    rules: [{
      test: /\.js?$/,
      exclude: /node_mudules/,
      loader: "babel-loader",
      query: {
        presets: ["@babel/preset-env"]
      },
    }],
  },
  mode: "development"
}
  1. Add script to run webpack
/* package.json */
"script": {
  "start": "webpack-dev-server --output-public-path=/build/",
  "build": "webpack"
}

  1. Add index.html which will contain build.js

  2. Add script tag for build.js

  3. Write your code.

  4. npm run build
